# ITSpend

## Overview
ITSpend is a data visualization project designed to provide clear insights into IT expenditure across business areas, cost elements, IT functions, and countries.  
It helps organizations track **actual vs forecast vs plan** spending, identify variances, and make informed financial decisions.

## Features
- 📊 Monthly trend analysis of forecast, actual, and planned IT spend  
- 🏢 Business area breakdown (Infrastructure, Manufacturing, Office & Admin, BU)  
- 💰 Cost element variance analysis (Salaries, Professional Services, Hardware, etc.)  
- 🌍 Country-wise expenditure variance tracking  
- ⚙️ IT area comparison (Functional, BU Support, Infrastructure, Enablement, Government)
